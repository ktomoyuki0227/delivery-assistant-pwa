# 📱 スマホでアクセスする方法

## 現在のサーバー情報

開発サーバーが起動しています!

### アクセスURL

- **PC**: http://localhost:3000
- **スマホ**: ターミナルの「Network」に表示されているURLを使用

ターミナル出力を確認して、以下のような行を探してください:

```
✓ Ready in 1899ms
- Local:   http://localhost:3000
- Network: http://192.168.x.x:3000  ← このURLをスマホで開く
```

## 手順

### 1. ネットワークURLを確認
ターミナルに表示されている **Network** のURLをメモ

### 2. スマホとPCを同じWi-Fiに接続
- PCとスマホが同じWi-Fiネットワークに接続されていることを確認

### 3. スマホのブラウザでアクセス
- Safari(iOS)またはChrome(Android)を開く
- ステップ1でメモしたURLにアクセス

### 4. カメラ権限を許可
- 初回アクセス時、カメラへのアクセスを許可

## トラブルシューティング

### アクセスできない場合

1. **ファイアウォールを確認**
   - Windowsセキュリティでポート3000を開放

2. **Wi-Fi接続を確認**
   - PCとスマホが同じネットワークにいるか確認

3. **開発サーバーを再起動**
   ```bash
   # Ctrl+C で停止
   npm run dev
   ```

詳しくは `MOBILE_TESTING.md` を参照してください!
